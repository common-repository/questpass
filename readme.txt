== Description ==

Thanks to Questpass, Publishers can finally fully monetize their potential, without overwhelming Users with a mass of traditional advertisements. They also become more independent from advertisers, without having to seek additional sponsored texts or affiliate programs.

What can you gain by joining Questpass? Lots of!

✔ Many times higher earnings than in other advertising systems.
✔ Effective monetization of all traffic, including mobile traffic!
✔ The ability to display questpasses also to adblock users!
✔ Additional earnings from paid subscriptions for content free from ads.

**[Find out more about Questpass »](https://questpass.pl/)**

== Installation ==

1. Install plugin through the WordPress plugins screen or download and install plugin directly from [Questpass Account Panel](https://system.questpass.pl/publisher/login).
2. Activate plugin through `Plugins` screen in WordPress Admin Panel.
3. Use `Settings -> Questpass` screen to configure the plugin.

If you need help or have questions, [please feel free to contact us](https://questpass.pl/contact).

= How to join Questpass? =

You need only three steps to start monetizing your content with Questpass:

1. Create a free Publisher's account in the Questpass system. [Create an account now »](https://system.questpass.pl/publisher/signup)

4. That's all. Now you can start earning.
